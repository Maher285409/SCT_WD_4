# SCT_WD_4

## 📝 To-Do Web Application

This project is a simple and efficient To-Do web application built using HTML, CSS, and JavaScript. It allows users to manage daily tasks by adding, editing, organizing, and tracking their completion status.

---

## 🚀 Features

* Add new tasks
* Mark tasks as completed
* Edit existing tasks
* Delete tasks
* Set date and time for tasks (if implemented)
* Organized task list view
* Responsive and user-friendly interface

---

## 🧠 How It Works

* Users can add tasks using an input field
* Each task is displayed in a list format
* Tasks can be marked as completed
* Users can edit or delete tasks anytime
* Date and time can be assigned to tasks (if added)

---

## 🛠️ Technologies Used

* HTML (structure)
* CSS (styling)
* JavaScript (task logic and interactivity)

---

## 📌 Project Goal

The goal of this project is to understand CRUD operations (Create, Read, Update, Delete), DOM manipulation, and building interactive web applications.
